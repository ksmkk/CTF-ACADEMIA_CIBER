Privilege Esc - Concepto SUID

Reto guiado: base tecnica del bit SUID

Objetivo

Entender que hace el bit SUID y sus aplicaciones.

Demo rapida

Binario comun:

-rwxr-xr-x 1 root root /usr/bin/tool

Binario con SUID:

-rwsr-xr-x 1 root root /usr/bin/tool

Cuando hay SUID, el binario corre con privilegios del owner (normalmente root).
